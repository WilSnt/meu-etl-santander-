# ETL Santander Dev Week - Projeto Básico

Este é um projeto simples de ETL feito como parte do desafio da DIO e Santander Dev Week.

O objetivo foi praticar:

- Extração de dados com Pandas
- Transformação simples (remoção de duplicados e preenchimento de valores)
- Criação de nova coluna
- Salvar dados tratados

## Como executar
```
pip install pandas
python etl_santander.py
```

Coloque o arquivo `usuarios_santander.csv` na mesma pasta antes de rodar.
